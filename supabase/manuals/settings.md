# Configurações — Perfil e Preferências

## O que é
A página de Configurações permite ao usuário gerenciar seu perfil pessoal, segurança da conta, preferências de notificação e aparência do sistema.

## Acesso
Menu lateral → (ícone de engrenagem) → Configurações
Ou acesse diretamente em /settings

## Seções Disponíveis

### 1. Perfil
| Campo | Descrição |
|-------|-----------|
| Nome | Nome de exibição no sistema |
| Email | Email de login (pode alterar) |
| Foto de Perfil | Avatar exibido no inbox |
| Cargo | Visualização do cargo atual (não editável aqui) |

### 2. Segurança
| Opção | Descrição |
|-------|-----------|
| Alterar Email | Troca o email de login |
| Alterar Senha | Define nova senha |
| Autenticação 2FA | Verificação em duas etapas (se disponível) |

### 3. Notificações
| Configuração | Descrição |
|-------------|-----------|
| Notificações Push | Alertas no navegador |
| Som de Notificação | Toque ao receber mensagem |
| Notificações por Email | Resumo por email |
| Novos Leads | Alerta quando um novo contato iniciar conversa |
| Menções | Alerta quando mencionado no chat interno |

### 4. Aparência
| Opção | Descrição |
|-------|-----------|
| Tema Claro/Escuro | Alternância de tema |
| Tema do Sistema | Segue preferência do SO |

### 5. PWA / Instalação
- Instale o Clinvia como aplicativo no celular ou desktop
- Clique em **"Instalar App"** e siga as instruções do navegador

### 6. Financeiro (Admin)
| Configuração | Descrição |
|-------------|-----------|
| Módulo Financeiro | Liga/desliga o módulo financeiro |
| Integração Bancária | Conecta conta bancária (se disponível) |

## Como Fazer

### Alterar Nome ou Foto
1. Acesse **Configurações → Perfil**
2. Edite o nome ou clique na foto para fazer upload
3. Clique em **Salvar**

### Alterar Senha
1. Acesse **Configurações → Segurança**
2. Clique em **"Alterar Senha"**
3. Digite a senha atual e a nova senha
4. Confirme e salve

### Ativar Notificações Push
1. Acesse **Configurações → Notificações**
2. Ative o toggle **"Notificações Push"**
3. Permita as notificações quando o navegador solicitar
4. Teste clicando em **"Testar Notificação"**

### Instalar como App (PWA)
1. Acesse **Configurações → PWA**
2. Clique em **"Instalar App"**
3. Confirme a instalação no banner do navegador
4. O app aparecerá na tela inicial do celular/desktop

## Problemas Comuns

| Problema | Solução |
|----------|---------|
| Notificações não chegam | Verifique se o navegador tem permissão; desative bloqueadores de notificação |
| Senha não aceita | Use no mínimo 8 caracteres com letras e números |
| Foto não carrega | Formatos aceitos: JPG, PNG. Tamanho máximo: 2MB |
| PWA não instala | Use Chrome ou Edge; Safari tem suporte limitado |

## Dicas
- Ative notificações push para não perder mensagens importantes
- Use tema escuro para reduzir o cansaço visual em longos turnos
- Instale o PWA para acesso mais rápido sem precisar abrir o navegador
